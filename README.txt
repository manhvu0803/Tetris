Requirement:
- Android Studio
- Android Emulator API level 28 / mobile device with Android 9.0 or higher

Compile and run
1. Open project with Android Studio (Open -> select this folder)
2. In Device Manager (top right), either
- Choose or create a virtual device with API level 28 higher
- Choose a physical device
2. Click Run 'app' (Shift + F10)
